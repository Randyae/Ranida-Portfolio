# Ranida Perera — Portfolio Website

Welcome to my personal portfolio website 🚀  
A modern, interactive, and visually engaging experience built to showcase my work and passion for web development.

🌐 **Live:** [ranida-portfolio.vercel.app](https://ranida-portfolio.vercel.app)

---

## 🌐 About

This portfolio represents my journey as a full-stack developer, born, raised, and based in **Colombo, Sri Lanka**. It's a central hub where you can explore my projects, learn about who I am, and get in touch.

---

## ✨ Features

- Smooth scroll-based animations and interactive UI
- Custom animated cursor with parallax effects
- Horizontal scroll "Get to Know Me" section with 3D panel hover effects
- Project showcase with live demos
- Manifesto section with word-reveal on scroll
- Contact section with live Colombo local time
- Light / Dark mode toggle
- Fully responsive design

---

## 🛠️ Tech Stack

- **React** + **Vite**
- **Vanilla CSS** (custom design system, no UI frameworks)
- **JavaScript** (Intersection Observer, requestAnimationFrame, CSS transforms)

---

## 🎯 Purpose

- Demonstrate my full-stack development skills
- Showcase real-world projects I've built
- Create a strong online presence for career opportunities

This project is deployed on **Vercel** for fast, reliable, and globally distributed hosting.
